# เนื้อเรื่องย่อ

เด็กชายคนหนึ่งเขาตื่นขึ้นที่หน้าดันเจี้ยนแห่งหนึ่ง เมื่อเดินเข้าไปเขาพบว่าตัวเองนั้นได้ติดอยู่ในดันเจี้ยนเขาวงกตแห่งนี้แล้ว ในมือของเขาได้มีเพียงคบไฟ และดาบข้างหน้าประตูที่เข้ามาเขาพบกระดาษใบหนึ่ง เขียนเอาไว้ว่า “ตามหาเหรียญทอง 20 เหรียญเพื่อเปิดประตู”เมื่อเด็กหนุ่มเห็นเช่นนั้นเขาจึงได้ตัดสินใจต้องสำรวจ ฟันฝ่าอุปสรรค และมอนเตอร์ต่างๆเข้าไปในดันเจี้ยนเพื่อตามหาเหรียญทั้ง 20 เหรียญที่ซ่อนอยู่ในดันเจี้ยนแห่งนี้

# รูปแบบการเล่น และ กติกา
	2D Top-Down RPG ผู้เล่นควบคุมตัวละครเดินซ้าย ขวา หน้า หลัง ต่อสู้กับศัตรูเพื่อเก็บเหรียญ  
	ประเภทของศัตรู ศัตรูมี 2 ประเภทคือ  
		- ศัตรูโจมตีใกล้ เดินเข้าหาผู้เล่นและฟันใส่ เมื่อโดนจะกระเด็นและพลังชีวิตลดลง  
		- ศัตรูโจมตีไกล เล็งผู้เล่นและยิงธนูใส่ผู้เล่น เมื่อโดนลูกธนูจะกระเด็นและพลังชีวิตลดลง  
	ระบบเหรียญ ผู้เล่นสามารถเก็บเหรียญหรือคะแนนจากจัดการศัตรูในแต่ละห้อง และเมื่อเก็บได้ครบจะสามารถหลบหนีและชนะเกมออกได้  

# แนวคิดการออกแบบ
	กราฟิก :  
	2D Pixel Art กราฟิกแบบพิกเซล ให้ความรู้สึกคลาสสิก มีความน่ารักในแบบฉบับของตนเอง เรียบง่ายแต่สวยงาม
	ระบบแสงเงา ในดันเจี้ยนที่มืดมิด มีเพียงแสงจากผู้เล่น และคบไฟเพียงเล็กน้อยในการนำทาง
	เสียง :  
	เพลงประกอบ เพลงหน้า Menuเข้าเกม  และในเกมทำนองความรู้สึกลึกลับและตื่นเต้นตลอดเวลา
	เสียงประกอบ เสียงโจมตีของผู้เล่นและศัตรู เสียงเก็บไอเท็ม เสียงโดนดาเมจ และเสียงตาย

# กลุ่มเป้าหมาย (ผู้เล่น)
	กลุ่มผู้เล่นทุกเพศทุกวัยที่มีความชื่นชอบการสำรวจ และต้องการความท้าทาย 

# การวิเคราะห์เกม AGE Analysis
	อารมณ์ของเกม:  
	ความเศร้า-เสียใจ เล่นแพ้ ตายก่อนเก็บเหรียญครบ  
	ความโกรธ จากการรับรู้ว่าต้องมาติดอยู่ในดันเจี้ยนและต้องเก็บเหรียญเท่านั้นถึงจะออกได้  
	ความตื่นเต้น การต้องเอาตัวรอดจากการโจมตีของศัตรูให้โดนดาเมจน้อยที่สุด และเก็บเหรียญให้ครบ  
	ความกลัว บรรยากาศของฉาก ที่มืดมิด และศัตรูที่ซ่อนในความมืด  
	ความภูมิใจ หลบลีกการโจมตี และกำจัดศัตรูที่มีความยากสูงได้  
	ความสุข  เก็บเหรียญจนครบและหลบหนีออกจากดันเจี้ยนได้  

# สิ่งที่ใช้กระตุ้นสัญชาติญาณ
	การเอาชีวิตรอด  
	การโจมตีของศัตรูที่เข้ามา  
	อยากสวมบทบาท (เป็นตัวละครในเกม)  
	ตัวละครเด็กหนุ่มไร้อารมณ์ ที่ไม่หวั่นเกรงต่อศัตรู  
	สะสมสิ่งของ  
	ระบบเก็บเหรียญ   
	ความโลภ  
	-  
	การปกป้อง  
	-  
	การสื่อสาร  
	-  
	ชอบสีสัน
	ออกแบบฉาก ใส่เสียงดนตรีให้มีสีสันแนวลึกลับ และน่าตื่นเต้น  
	สำรวจ  
	มีห้องที่หลากหลายให้สำรวจ  
	ก้าวร้าว  
	มีระบบการต่อสู้กำจัดศัตรู  
	ความแค้น  
	ศัตรูมีความแข็งแกร่งที่สูง เลือดผู้เล่นต่ำ ต้องใช้ความสามารถในการหลบลีกการโจมตีของศัตรู ทำให้ผู้เล่นพ่ายแพ้ง่าย  
	แข่งขัน  
	ระบบเหรียญ จากการกำจัดศัตรูและจัดลำดับเวลาเล่นเลียร์แต่ละครั้ง  

# ประโยชน์ของเกม
	ช่วยพัฒนาทักษะการควบคุมและการตอบสนอง ฝึกไหวหริบ ความตื่นตัวในการต่อสู้กับศัตรูไม่ให้ตายก่อนเก็บเหรียญครบ 
	ฝึกการวางแผน วางแผนในการต่อสู้เพื่อให้ไปเก็บเหรียญ จัดการศัตรูโดยเสียเลือดน้อยที่สุด ไม่ตาย และเก็บเหรียญครบ
	ฝึกความอดทน ในการเล่นจนผ่าน ต้องใช้ความอดทนสูง
